# 🏠 Real Estate ML App

An end-to-end Machine Learning application that provides **Price Prediction**, **Data Analysis**, and **Apartment Recommendation System** for real estate properties.

---

## 🚀 Features

### 📊 1. Data Analysis

* Exploratory Data Analysis (EDA)
* Visual insights on pricing, location, and features
* Interactive charts and graphs

### 💰 2. Price Prediction

* Predict property prices based on user inputs
* Built using machine learning pipeline
* Real-time predictions via Streamlit UI

### 🤝 3. Recommendation System

* Suggest similar apartments based on features
* Uses cosine similarity for recommendations
* Distance-based filtering included

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn, Plotly
